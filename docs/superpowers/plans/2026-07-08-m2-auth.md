# M2 ÔÇö Authentication Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Session-cookie authentication for the Flask API ÔÇö login/logout/me/csrf endpoints backed by bcrypt, account lockout, and Flask-Login, with CSRF protection and hardened cookies.

**Architecture:** A pure `auth_service.authenticate()` holds the credential/lockout logic (tested against the DB, no HTTP). Flask-Login manages server-side sessions; the `User` model becomes a `UserMixin`. Flask-WTF provides CSRF. A thin `auth` blueprint exposes the endpoints and returns JSON (401 on unauthenticated, never a redirect). The React login UI arrives in M3 and consumes these endpoints.

**Tech Stack:** Flask-Login, Flask-WTF (CSRF), bcrypt (from M1's `security.py`), pytest. Builds on the M1 backend.

## Global Constraints

- Inherits all M1 constraints (Python 3.11+, Flask 3.x, SQLAlchemy 2.x, SQLite dev/test + SQL Server prod, roles as JSON string, statuses as strings, brand-color-only UI).
- **Auth model:** Flask-Login server-side sessions. Cookies **httpOnly + sameSite=Lax**, **secure in prod**. **CSRF protection** on all mutating requests via a token from `GET /api/auth/csrf`.
- **Lockout:** after **5** consecutive failed logins, lock for **15 minutes**. Reset the counter on success.
- **Username/email are case-insensitive** (stored lowercased). Unknown-user logins do a dummy bcrypt compare to equalize response timing (no user enumeration).
- All auth routes are under `/api/auth`.

`backend/app/services/auth_service.py`:
```python
from dataclasses import dataclass
from datetime import datetime, timedelta, timezone
from typing import Optional

from app.extensions import db
from app.models import User
from app.services.security import verify_password, hash_password

MAX_FAILED_LOGINS = 5
LOCKOUT_MINUTES = 15

# Precomputed hash used only to equalize timing for unknown usernames.
_DUMMY_HASH = hash_password("timing-equalization-placeholder")


@dataclass
class AuthResult:
    ok: bool
    user: Optional[User] = None
    error: Optional[str] = None


def _now() -> datetime:
    return datetime.now(timezone.utc)


def _as_aware(dt: datetime) -> datetime:
    # SQLite returns naive datetimes; treat stored timestamps as UTC so
    # comparisons against an aware "now" don't raise TypeError.
    return dt if dt.tzinfo is not None else dt.replace(tzinfo=timezone.utc)


def authenticate(username: str, password: str) -> AuthResult:
    uname = (username or "").strip().lower()
    user = db.session.query(User).filter_by(username=uname).one_or_none()

    if user is None:
        verify_password(password, _DUMMY_HASH)  # equalize timing
        return AuthResult(ok=False, error="Invalid username or password.")

    if user.locked_until is not None and _as_aware(user.locked_until) > _now():
        return AuthResult(ok=False, error="Account locked. Try again later.")

    if not user.active:
        return AuthResult(ok=False, error="Account is inactive.")

    if not verify_password(password, user.password_hash):
        user.failed_logins += 1
        if user.failed_logins >= MAX_FAILED_LOGINS:
            user.locked_until = _now() + timedelta(minutes=LOCKOUT_MINUTES)
        db.session.commit()
        return AuthResult(ok=False, error="Invalid username or password.")

    user.failed_logins = 0
    user.locked_until = None
    db.session.commit()
    return AuthResult(ok=True, user=user)
```

```python
from flask import Blueprint, jsonify, request
from flask_login import login_user, logout_user, current_user, login_required
from flask_wtf.csrf import generate_csrf

from app.services.auth_service import authenticate

bp = Blueprint("auth", __name__, url_prefix="/api/auth")


def _user_json(user):
    return {
        "id": user.id,
        "username": user.username,
        "name": user.name,
        "email": user.email,
        "roles": user.roles_list,
    }


@bp.get("/csrf")
def csrf_token():
    return jsonify(csrfToken=generate_csrf())


@bp.post("/login")
def login():
    data = request.get_json(silent=True) or {}
    result = authenticate(data.get("username", ""), data.get("password", ""))
    if not result.ok:
        return jsonify(error=result.error), 401
    login_user(result.user)
    return jsonify(_user_json(result.user))


@bp.post("/logout")
@login_required
def logout():
    logout_user()
    return jsonify(ok=True)


@bp.get("/me")
@login_required
def me():
    return jsonify(_user_json(current_user))
```
